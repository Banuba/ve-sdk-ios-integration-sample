
### 1.0.17

*Released on July 12, 2021*

This release requires prior migration to **0.0.16.3**

## List of changes

1. Save undone editing progress to **Drafts**. Continue making cool videos anytime!

    <img src="https://github.com/Banuba/ve-sdk-ios-integration-sample/blob/main/mdDocs/gif/drafts.gif" alt="Screenshot" height="auto">&nbsp;

2. New Picture in Picture modes: React, Top-Bottom, Left-Right:
     
     <img src="https://github.com/Banuba/ve-sdk-ios-integration-sample/blob/main/mdDocs/gif/PiP_all.gif" alt="Screenshot" height="auto">&nbsp;

3. Export performance optimized by 15-60% (see details in the performance section below)
4. You can configure **muting microphone behavior** on camera screen for some scenarios with the help of `CameraMuteMicConfig` class:

    ```swift
    struct RecorderConfiguration {
    		...
        /// Mute microphone on PIP screen. Default is false.
        public var muteMicrophoneForPIP: Bool
    }
    ```

    To setup custom configuration provide following code:

    ```swift
    let config = VideoEditorConfig()
    config.recorderConfiguration.muteMicrophoneForPIP = true
    ```

5. The SDK now collects **few more data:** start and initiation of the video editor,  exported video metadata. This includes the following data:
    1. export duration
    2. length of the exported video
    3. number of exported videos
    4. resolution
    5. used effects
    6. SDK version
    7. OS version
    8. Device
    9. Token

    **SDK does not collect any user data!**

6. Main Bugfixes. 
    - Improve PIP Audio functionality
    - Remove cached Giphy API key(giphyApiKey)
    - Fix handsfree usage with specific iOS version
    - Fix cursor position after recording audio
    - The thumbnails on the add effect screens match the preview
    - The track is looped correctly on post-processing
    - Redo the logic for setting the image for the preview:

        The name of the icon resource must be the same as the graphic file ID the / luts directory.   See the [guide](https://github.com/Banuba/ve-sdk-ios-integration-sample/blob/main/mdDocs/faq.md#5-how-do-i-add-a-color-filter-lut).

## Migration guide

1. **PIP - picture in picture.**

    To be able to use the newest PiP modes you need to adjust layoutSettingsButtonsConfiguration option.

    ```swift
    /// The PIP Setting Configuration
    public struct PIPSettingsConfiguration {
      /// BackgroundConfiguration setups background view style
      public var backgroundConfiguration: BackgroundConfiguration
      /// Cursor color
      public var dragIndicatorConfiguration: RoundedButtonConfiguration
      /// Title font for controls
      public var titleConfiguration: TextConfiguration
      /// Main settings buttons' configurations
      public var layoutSettingsButtonsConfiguration: [PIPSelectableCellConfiguratn]
    }
    ```

    PIPSelectableCellConfiguration consists of several parameters. 

    ```swift
    public struct PIPSelectableCellConfiguration {
      /// The button Identifier
      public var identifier: PIPPlayerLayoutSetting
      /// The PIP selectable cell switch config
      public var switchCellConfiguration: PIPSwitchCellConfiguration?
      /// The PIP selectable cell camera config
      public var cameraCellConfiguration: PIPCameraCellConfiguration
      /// The button text configuration.
      public var textConfiguration: TextConfiguration
      /// The default image configuration
      public var imageConfiguration: ImageConfiguration
      /// The border width configuration.
      public var borderWidth: CGFloat
      /// The border color configuration.
      public var borderColor: CGColor
      /// Is image view circable
      public var isRoundedImageView: Bool
      /// The corner radius configuration
      public var cornerRadius: CGFloat?
      /// The background color configuration.
      public var backgroundColor: UIColor
      /// Additional button width.
      ///Default is 130
      public var additionalButtonWidth: CGFloat
      /// Additional button height.
      ///Default is 32
      public var additionalButtonHeight: CGFloat
    }
    ```

    The newest PiP modes describes into PIPPlayerLayoutSettings entity.

    ```swift
    // MARK: - PIPPlayerLayoutSettings
    @objc public enum PIPPlayerLayoutSettings: Int {
      case floating
      case topBottom
      case react
      case leftRight
    }
    ```

2. **Drafts**

    Drafts usage extends existing restoring session functionality

    To override DraftsConfiguration entity use options below. 

    DraftsConfiguration is a part of VideoEditorConfig.

    ```swift
    /// Drafts screen configuration
    public struct DraftsConfiguration {
      /// BackButtonConfiguration setups back button style
      public var backButton: BackButtonConfiguration
      /// TextConfiguration setups title style
      public var title: TextConfiguration
      /// DraftItemConfiguration setups draft item style
      public var draftItem: DraftItemConfiguration
      /// Setups image for empty data source
      public var emptyDraftsImage: ImageConfiguration
      /// Setups label for empty data source
      public var emptyDraftsLabel: TextConfiguration
    }
    ```

3. **API Changes**

    We have already deprecated SingleTrimConfiguration entity. You need to remove the usage of this configuration.

    ```swift
    ~~config.singleTrimConfiguration = ...~~ // Delete this line if exist
    ```
